# 🛡️ Cybersecurity & Safety Check Tool

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

**Safety Check Tool** — это кроссплатформенный комплекс инструментов для обеспечения личной цифровой безопасности и проверки контактов. Проект разрабатывается совместно **Salmonellababa** и **Beluga** для защиты пользователей от мобильного мошенничества и сетевых угроз.

---

## 📱 Поддерживаемые платформы

### 1. 🤖 Версия для Android (Основной фокус)
*   **Формат**: Полноценное мобильное приложение (**APK**).
*   **Сборка**: Автоматизирована и компилируется с помощью системы сборки **Gradle**.
*   **Функционал**: Быстрый чекер подозрительных контактов прямо на смартфоне, анализ спам-номеров перед звонком и защита от фишинга.

### 2. 🪟 Версия для Windows
*   ⏳ **Coming Soon** (В разработке). Настольная версия со сканером процессов и интеграцией в систему появится в следующих релизах.

## ⚙️ Основной функционал (Android App)

*   **📞 Safety Check (Проверка контактов)**: Быстрый анализ телефонных номеров на предмет спам-меток, скрытых тегов и мошеннической активности.
*   **🛡️ Антивирусный контур**: Интеграция легковесных сканеров для проверки подозрительных файлов и входящих ссылок.
*   **🔒 Конфиденциальность**: Все проверки происходят в изолированном режиме без передачи персональных данных пользователей третьим лицам.
